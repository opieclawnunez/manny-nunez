# manny-nunez

Professional landing page for Manuel Nunez, MD — Family Medicine Physician transitioning from military service to outpatient practice in Arizona.

## Stack

- Static HTML + CSS via Tailwind CDN
- Zero build step, zero dependencies
- Deployed on Vercel

## Sections

1. **Hero** — Name, title, location, availability badge
2. **About** — Background and transition narrative
3. **Credentials** — Boards, licenses, DEA, education
4. **What I'm Looking For** — Outpatient FM, Litchfield Park AZ, FTE flexibility, low call
5. **Contact** — Email and LinkedIn links

## Edit

Open `index.html` in any editor. The Tailwind CDN handles all styling. Push to `main` on GitHub to auto-deploy via Vercel.
